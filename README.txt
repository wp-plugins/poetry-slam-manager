Thanks for downloading Slam Manager!

How it works:

1. Cut and paste the shortcode '[slam-display-results]' into a post/page.
2. View the page while logged in as an admin.  You should be able to create and edit slams from here.

How to install (if you haven't already):



Upload a ZIP file of the plugin to your WordPress site through the Plugins section of your site.
Hit the Install Button.
Wait.
It should work now.
